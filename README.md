# Steviggio Next.js Portfolio

- Next.js + TypeScript
- TailwindCSS
- Lucide-react

## Description

This is a Next.js portfolio with a simple design, which can be used as a template for a fast deployment.

## Installation

Clone the repo and install dependencies :

```bash
git clone https://github.com/Steviggio/next-portfolio.git
cd next-portfolio
npm install
```

## Usage

Run the app :

```bash
npm run dev
```
